Wallet Connect Example

This is an example page showing how to integrate wallet connection functionality into your custom design. Use this as a reference for your own implementation.

Click below to test if it works!

Setup Instructions

  1. Create your own HTML page with your design
  2. Add a button with ID connect-wallet (or any ID you prefer)
  3. Include the loader script: <script src="loader.js"></script>
  4. The wallet modal will automatically work with buttons that have the connect-wallet ID
  5. You can also use connect-wallet-hero or any other ID - just update the script accordingly
